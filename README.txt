Rock, Paper, Scissors Game in Python
------------------------------------

This repository contains a Python implementation of the classic game Rock, paper, scissors. It is a command-line based game where a user competes against the computer.

Gameplay
--------

The game follows the traditional rules of rock, paper, scissors:

- Rock beats scissors
- Scissors beat paper
- Paper beats rock

The user and the computer play rounds until one of them wins twice. Each round, the user inputs their choice, and the computer makes a random selection. After each round, the current score is displayed.

Running the Game
----------------

To run the game, you will need Python 3. Use the following command in the terminal to start the game:

python rock_paper_scissors.py

This is a simple implementation, designed as a beginner's project. Any feedback or suggestions for improvements are welcome.
